## About Osta (أُسطى)

**Osta** is a home services marketplace platform that connects clients with verified local technicians across a range of service categories — from plumbing and electrical work to emergency and utilities requests.

This repository contains the **Next.js frontend**, built with TypeScript and Tailwind CSS, and designed to serve three user roles:

- **Clients** — browse services, submit requests, track job status, and leave reviews
- **Technicians** — manage incoming requests, update availability, and view earnings
- **Admins** — oversee the platform, manage users, categories, and service operations

### Tech Stack
- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **JWT-based authentication** (with Google OAuth support)
- **Paymob** payment gateway integration

### Backend
The backend API is built with **NestJS + MongoDB**. See [osta-backend](link-to-backend-repo) for the API source.
